# Indian Cyber Crime Dashboard

The **Indian Cyber Crime Dashboard** is a data visualization project that provides insights into cyber crime trends across different states of India. The dashboard presents a comprehensive overview of cyber crime statistics from 2016 to 2018, helping users understand patterns, trends, and the impact of cyber crime relative to population and state demographics.

## Features
- **State-wise Cyber Crime Statistics**: Visualizes the number of reported cyber crimes per state.
- **Percentage Share Analysis**: Shows the contribution of each state to the national cyber crime data.
- **Population vs Crime Rate Analysis**: Compares cyber crime incidence with population size to identify high-risk regions.
- **Interactive Charts & Graphs**: Makes complex data easier to understand.

## Objective
The project aims to:
- Help analysts and researchers understand cyber crime distribution in India.
- Support data-driven decision making for policymakers and law enforcement agencies.
- Serve as a reference for students and professionals interested in data visualization and cyber security analytics.
